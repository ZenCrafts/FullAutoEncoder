■ 概要
・この度は、本ソフトをご利用いただきありがとうございます。
　本ソフトは、フォルダを監視し、動画を「丸投げ」で変換・整理するためのツールです。
　Windows 11での動作を確認しています。
　※本アプリはAI（Gemini）と相談しながら、約2ヶ月の試行錯誤を経て制作いたしました。

■ 主な特徴
・監視フォルダ（例 E:\rec_result）に動画が追加されると、自動で出力フォルダへmp4出力します。
・録画ソフト（EpgTimer等）の保存先を監視するだけで、全自動エンコード環境が構築できます。
・対応形式：.ts .mkv .avi .ogm .mp4 .wmv .mov .mpg .mpeg .flv .webm .asf .vob .m2ts
・スリープ抑制機能：エンコード中はPCのスリープを防止し、完了後に自動でスリープ移行します。
・2本並行エンコード：PCスペックを活かし、トータルの処理時間を大幅に短縮可能です。

■ 同梱ファイル
・FullAutoEncoder.exe (本体)

■ インストール方法
・本ソフトはインストール不要です。ZIPを解凍し、フォルダごと任意の場所に配置してください。
・【推奨配置例】C:\Tools\FullAutoEncoder\
　※ C:\Program Files 内では、権限の関係で設定保存が失敗する場合があります。

■ 準備するもの (Prerequisites)
・FFmpeg.exe / ffprobe.exe
　公式サイト（https://ffmpeg.org）等から入手し、
　FullAutoEncoder.exe と同じフォルダに配置してください。

■ 基本的な使い方 (Basic Usage)
1. アプリを起動し、メニューの「設定」から画質や使用GPU、作業パスを設定します。
2. メイン画面で「監視フォルダ」と「出力フォルダ」を指定します。
3. 「スキャン＆Encode開始」で監視を開始します（既存ファイルも処理されます）。
4. 出力ファイル名には [再生時間/解像度/コーデック] が自動付与されます。
5. 処理済み動画は、監視フォルダ内の「エンコード済み」フォルダへ自動移動します。

■ 補足・制限事項
・.vobファイルは構造上、進捗バーが正しく表示されませんが、エンコードは正常に行われます。
・動作ログは同フォルダの operation_log.txt に記録されます。
